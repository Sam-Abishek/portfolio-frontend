IoT Cyber Attack Detection Using Machine Learning

This project focuses on enhancing the security of IoT devices by implementing a machine learning-based detection system. The key features include:

  • Data collection from IoT networks to capture normal and malicious traffic.
  • Data preprocessing and feature engineering.
  • Training ML algorithms like Random Forest, SVM, and Neural Networks.
  • Real-time anomaly detection for early threat identification.
  • Visualization of attack patterns and system alerts.

Outcome: Increased resilience of IoT ecosystems against cyber attacks and data breaches.
