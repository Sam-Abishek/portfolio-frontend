Web Application for Asset Management

This project is a full-stack web application designed to manage organizational assets efficiently. It includes:

  • Asset entry system for adding new assets with details.
  • Category-based sorting and filtering.
  • Dynamic reporting and asset tracking.
  • User authentication for secure access.
  • Clean and responsive user interface built with HTML, CSS, and JavaScript.

Outcome: Streamlined asset management process, improved accountability, and simplified reporting for organizations.
